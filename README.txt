Allow your WooCommerce website to send SMS notifications to your customers.


== Description ==

Keep customers updated with automated SMS for order notifications

Current features:

*  Auto send status updates
*  Different SMS send corresponding to different WooCommerce Order Status
*  Customizable message variables using {{shop_name}}, {{order_id}}, {{order_amount}}, {{order_status}}
*  Customizable message text


= Requirements =

A ShoutOUT Lite account.


== Installation ==


= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "ShoutOUT SMS for WooCommerce" and click Search Plugins. Once you've found our plugin you can view details about it such as the the rating and description. Most importantly, of course, you can install it by simply clicking Install Now.


= Manual Installation =

1.  Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting previous versions if they exist
2.  Activate the plugin in your WordPress admin area.

or

1.  Log into WordPress Dashboard , navigate to the Plugins menu and click Add New.
2.  Click "Upload Plugin" then chose "shoutout_sms_for_woocommerce.zip" file. Next click "Install Now".
3.  Activate the plugin in your WordPress admin area.


= configuration =

Easy steps to install the plugin:

*	Upload "shoutout_sms_for_woocommerce" folder/directory to the /wp-content/plugins/ directory
*	Activate the plugin through the 'Plugins' menu in WordPress.
